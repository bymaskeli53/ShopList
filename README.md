## üåü Overview

ShopList is a **Kotlin Multiplatform** shopping list application that demonstrates modern Android development practices with **MVVM architecture**, **Material Design 3**, and **accessibility-first** design principles. Built with Compose Multiplatform, it runs natively on Android and iOS with shared business logic.

### Why ShopList?

- üé® **Beautiful UI**: High-contrast design with Material You dynamic colors (Android 12+)
- ‚ôø **Accessibility First**: Large touch targets (48dp+), screen reader support, and TTS features
- üîÑ **Reactive Architecture**: Built with Kotlin Flow and StateFlow for seamless data updates
- üéØ **Clean Code**: MVVM architecture with clear separation of concerns
- üì± **Multiplatform**: Shared codebase for Android and iOS
- üóÉÔ∏è **Offline First**: Local SQLDelight database with instant data persistence
- üåô **Dark Mode**: High-contrast dark theme optimized for readability

## üöÄ Getting Started

### Prerequisites

- **Android Studio** Ladybug | 2024.2.1 or newer
- **JDK** 17 or newer
- **Kotlin** 2.2.0
- **Gradle** 8.11.1
- **Xcode** 15+ (for iOS development)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/shoplist.git
   cd shoplist
   ```

2. **Open in Android Studio**
   - Launch Android Studio
   - Select "Open an Existing Project"
   - Navigate to the cloned directory

3. **Sync Gradle**
   ```bash
   ./gradlew build
   ```

4. **Run the app**
   - **Android**: Click "Run" or press `Shift + F10`
   - **iOS**: Open `iosApp` in Xcode and run
